# Show Hide Animation CSS
Est une animation qui permet d'afficher et de faire disparaitre un élément en utilisant **HTML** et **CSS** uniquement.

## Cas d'utilisation
1. CSS Grid
2. Positionnement **Relative** et **Absolute**
3. Label et CheckBox
4. _@keyframes
